## [0.1.0] - 2024-XX-XX

### Added
- Initial release
- Personal Access Token authentication
- OAuth2 authentication
- Person resource with CRUD operations
  - Create person
  - Get person by ID
  - Get many people with filters
  - Update person
  - Delete person
- JSON-API request/response handling
- Pagination support
- Include related resources (emails, phones, addresses, etc.)
